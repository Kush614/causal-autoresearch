Causal Verification Pipeline

The complete flow from git commit to causal classification. Each commit triggers the interceptor, which runs 3 independent tests and combines scores.

Git Commit
Agent edits train.py
→
Interceptor
Reads val_bpb + diff
→
Test A: Ablation
Reverse patch, retrain
→
Test B: Replication
3 seeds, measure var
→
Test C: Transfer
Compare to baseline
→
Classifier
Weighted score → label

Causal Graph (Pearl's Do-Calculus)

The ratchet measures P(Y|X) = correlation. Our layer measures P(Y|do(X)) = causation by controlling for confounders.

Causal Graph: Random Seed (Z1) ──────┐ v Code Change (X) ────> val_bpb (Y) ^ Data Order (Z2) ──────┘ # Standard ratchet: P(Y | X) = Correlation # Our causal layer: P(Y | do(X)) = Causation # Test A controls for: do(X=0) intervention # Test B controls for: Marginalize over Z1 (seed) # Test C controls for: Generalizability across environments

causal/interceptor.py

Entry point for causal verification. Called with a commit hash, orchestrates all 3 tests, computes the final score, and logs to causal_results.tsv.

from causal.ablation import run_ablation from causal.replication import run_replication from causal.transfer import run_transfer from causal.classifier import compute_causal_score, classify def intercept_commit(commit_hash): # Get current val_bpb from run.log val_bpb = get_val_bpb_from_log() # Get the git diff for this commit diff = get_commit_diff(commit_hash) # Run three independent causal tests ablation = run_ablation(diff, val_bpb) replication = run_replication(val_bpb) transfer = run_transfer(val_bpb) # Compute weighted score + classify causal_score = compute_causal_score(ablation, replication, transfer) classification = classify(causal_score) # Log to causal_results.tsv log_result(commit_hash, val_bpb, ablation, replication, transfer, causal_score, classification) return causal_score

Helper: Reading val_bpb

def get_val_bpb_from_log(): """Read val_bpb from most recent run.log""" with open('run.log', 'r') as f: for line in f: if line.startswith('val_bpb:'): return float(line.split(':')[1].strip())

Helper: Getting Commit Diff

def get_commit_diff(commit_hash): """Get the diff of train.py changes in this commit""" result = subprocess.run( ['git', 'diff', f'{commit_hash}^', commit_hash, '--', 'train.py'], capture_output=True, text=True ) return result.stdout

causal/ablation.py — Test A (Weight: 50%)

Reverses the agent's git diff on train.py, re-runs training, and measures the delta. If val_bpb gets worse without the change, the change was genuinely causal. This is the do(X=0) intervention from Pearl's framework.

NORMALIZATION_DELTA = 0.005 # 0.005 val_bpb improvement = score of 1.0 def run_ablation(diff, current_val_bpb): """Reverse the change, re-run, measure delta.""" # Save current train.py with open('train.py', 'r', encoding='utf-8') as f: original_content = f.read() try: # Apply reverse patch (remove the agent's change) success = apply_reverse_patch(diff) if not success: return 0.5 # Can't reverse cleanly # Run training WITHOUT the change ablated_bpb = run_training_get_bpb() finally: # Always restore original train.py with open('train.py', 'w', encoding='utf-8') as f: f.write(original_content) # Without the change, val_bpb should be HIGHER (worse) improvement = ablated_bpb - current_val_bpb # positive = causal # Normalize to 0-1 score = min(1.0, max(0.0, improvement / NORMALIZATION_DELTA)) return score
def apply_reverse_patch(diff): """Apply reverse of diff to train.py""" result = subprocess.run( ['git', 'apply', '--reverse', '--whitespace=fix', '-'], input=diff, text=True, capture_output=True ) return result.returncode == 0

causal/replication.py — Test B (Weight: 30%)

Runs training 3 times with different random seeds [42, 137, 999]. Low variance across seeds means the improvement is seed-independent and likely causal. High variance means the improvement was a lucky seed.

SEEDS = [42, 137, 999] VARIANCE_THRESHOLD = 0.002 # variance above this = fully spurious def run_training_with_seed(seed): """Temporarily inject seed into train.py, run, return val_bpb.""" with open('train.py', 'r', encoding='utf-8') as f: content = f.read() # Inject seed at top of file (after imports) seed_injection = f""" import random; random.seed({seed}) import numpy; numpy.random.seed({seed}) import torch; torch.manual_seed({seed}) torch.cuda.manual_seed_all({seed}) """ # Write modified, run training, then ALWAYS restore try: with open('train.py', 'w', encoding='utf-8') as f: f.write(modified_content) bpb = run_training_get_bpb() finally: with open('train.py', 'w', encoding='utf-8') as f: f.write(content) # Always restore original return bpb
def run_replication(reference_val_bpb): """Run 3 seeds, measure variance. Returns score 0.0-1.0""" results = [] for seed in SEEDS: bpb = run_training_with_seed(seed) if bpb is not None: results.append(bpb) import statistics variance = statistics.variance(results) # Low variance = high score (seed-independent) score = max(0.0, 1.0 - (variance / VARIANCE_THRESHOLD)) return min(1.0, score)

causal/transfer.py — Test C (Weight: 20%)

Re-runs training and compares against a stored baseline. If the improvement holds across different conditions, it generalizes and is likely causal. Uses smoothed scoring thresholds.

def run_transfer(current_val_bpb): """Check if improvement holds on different evaluation.""" # Run training one more time transfer_bpb = run_training_get_bpb() # Load stored baseline from before any agent commits with open('causal/transfer_baseline.txt') as f: baseline_bpb = float(f.read().strip()) improvement = baseline_bpb - transfer_bpb # positive = still improving # Smoothed scoring if improvement > 0.001: return 1.0 # Clear improvement elif improvement > 0: return 0.7 # Marginal elif improvement > -0.001: return 0.3 # Flat else: return 0.0 # Regression

causal/classifier.py — Score Combiner

Combines the three test scores using weighted averaging and classifies the result into one of four tiers. This is the final step in the causal verification pipeline.

WEIGHTS = { 'ablation': 0.50, 'replication': 0.30, 'transfer': 0.20, } THRESHOLDS = { 'strongly_causal': 0.80, 'likely_causal': 0.50, 'uncertain': 0.20, # below 0.20 -> spurious } def compute_causal_score(ablation, replication, transfer): """Compute weighted causal confidence score.""" score = ( WEIGHTS['ablation'] * ablation + WEIGHTS['replication'] * replication + WEIGHTS['transfer'] * transfer ) return round(min(1.0, max(0.0, score)), 3) def classify(score): """Map confidence score to classification label.""" if score >= THRESHOLDS['strongly_causal']: return 'strongly_causal' elif score >= THRESHOLDS['likely_causal']: return 'likely_causal' elif score >= THRESHOLDS['uncertain']: return 'uncertain' else: return 'spurious'
